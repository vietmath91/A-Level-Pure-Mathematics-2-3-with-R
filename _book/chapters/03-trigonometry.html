4.1 Overview

This chapter develops five connected ideas:

  1. the cosecant, secant and cotangent ratios;
  2. compound angle formulae;
  3. double angle formulae;
  4. further trigonometric identities;
  5. expressing \(a\sin\theta+b\cos\theta\) in the form \(R\sin(\theta\pm\alpha)\) or \(R\cos(\theta\pm\alpha)\).

The central theme is that trigonometric expressions can often be transformed into simpler or more useful forms. These transformations are essential for solving equations, proving identities and preparing for later work in differentiation and integration.

4.2 3.1 The cosecant, secant and cotangent ratios

4.2.1 Key mathematical ideas

The three reciprocal trigonometric functions are defined by

\[ \operatorname{cosec}\theta=\dfrac{1}{\sin\theta}, \]

\[ \sec\theta=\dfrac{1}{\cos\theta}, \]

and

\[ \cot\theta=\dfrac{1}{\tan\theta}. \]

Since

\[ \tan\theta=\dfrac{\sin\theta}{\cos\theta}, \]

we also have

\[ \cot\theta=\dfrac{\cos\theta}{\sin\theta}. \]

These definitions immediately show where the reciprocal functions are undefined.

The function \(\operatorname{cosec}\theta\) is undefined when

\[ \sin\theta=0. \]

The function \(\sec\theta\) is undefined when

\[ \cos\theta=0. \]

The function \(\cot\theta\) is undefined when

\[ \sin\theta=0. \]

4.2.2 Visualisation using R

Code 
theta <- seq(0, 360, length.out = 3000)

df <- data.frame(
  theta = rep(theta, 3),
  function_name = rep(c("cosec(theta)", "sec(theta)", "cot(theta)"), each = length(theta))
)

df$y <- ifelse(
  df$function_name == "cosec(theta)",
  1 / sin_deg(df$theta),
  ifelse(
    df$function_name == "sec(theta)",
    1 / cos_deg(df$theta),
    1 / tan_deg(df$theta)
  )
)

df$y[abs(df$y) > 8] <- NA

ggplot(df, aes(x = theta, y = y)) +
  geom_line(linewidth = 1.0, na.rm = TRUE) +
  facet_wrap(~ function_name, ncol = 1) +
  geom_hline(yintercept = 0) +
  coord_cartesian(ylim = c(-8, 8)) +
  labs(
    title = "Reciprocal trigonometric functions",
    subtitle = "Large values near vertical asymptotes have been omitted for readability.",
    x = "theta in degrees",
    y = "y"
  ) +
  book_theme()

The vertical asymptotes occur where the denominator of the reciprocal function is zero.

4.2.3 Worked example

Find the exact value of

\[ \operatorname{cosec}240^\circ. \]

Since

\[ \operatorname{cosec}\theta=\dfrac{1}{\sin\theta}, \]

we have

\[ \operatorname{cosec}240^\circ=\dfrac{1}{\sin240^\circ}. \]

Now \(240^\circ\) is in the third quadrant, where sine is negative. Its reference angle is \(60^\circ\), so

\[ \sin240^\circ=-\sin60^\circ. \]

Therefore,

\[ \sin240^\circ=-\dfrac{\sqrt3}{2}. \]

Hence,

\[ \operatorname{cosec}240^\circ=-\dfrac{2}{\sqrt3} = -\dfrac{2\sqrt3}{3}. \]

4.3 3.2 Compound angle formulae

4.3.1 Key mathematical ideas

The compound angle formulae allow us to expand trigonometric functions of sums and differences.

For sine,

\[ \sin(A+B)\equiv\sin A\cos B+\cos A\sin B, \]

and

\[ \sin(A-B)\equiv\sin A\cos B-\cos A\sin B. \]

For cosine,

\[ \cos(A+B)\equiv\cos A\cos B-\sin A\sin B, \]

and

\[ \cos(A-B)\equiv\cos A\cos B+\sin A\sin B. \]

For tangent,

\[ \tan(A+B)\equiv \dfrac{\tan A+\tan B}{1-\tan A\tan B}, \]

and

\[ \tan(A-B)\equiv \dfrac{\tan A-\tan B}{1+\tan A\tan B}. \]

4.3.2 Visualisation using R

Code 
A <- seq(0, 360, length.out = 800)
B <- 40

df <- data.frame(
  A = A
)

df$left_side <- sin_deg(df$A + B)
df$right_side <- sin_deg(df$A) * cos_deg(B) + cos_deg(df$A) * sin_deg(B)

ggplot(df, aes(x = A)) +
  geom_line(aes(y = left_side, linetype = "sin(A + B)"), linewidth = 1.1) +
  geom_line(aes(y = right_side, linetype = "sin A cos B + cos A sin B"), linewidth = 1.1) +
  geom_hline(yintercept = 0) +
  labs(
    title = "Compound angle formula for sine",
    subtitle = "The two graphs overlap because the expressions are identical.",
    x = "A in degrees",
    y = "value",
    linetype = "Expression"
  ) +
  book_theme()

4.3.3 Worked example

Find the exact value of

\[ \sin75^\circ. \]

Write

\[ 75^\circ=45^\circ+30^\circ. \]

Then

\[ \sin75^\circ=\sin(45^\circ+30^\circ). \]

Using the compound angle formula,

\[ \sin(45^\circ+30^\circ) = \sin45^\circ\cos30^\circ+\cos45^\circ\sin30^\circ. \]

Substituting exact values,

\[ \sin75^\circ = \dfrac{\sqrt2}{2}\cdot\dfrac{\sqrt3}{2} + \dfrac{\sqrt2}{2}\cdot\dfrac{1}{2}. \]

Therefore,

\[ \sin75^\circ = \dfrac{\sqrt6+\sqrt2}{4}. \]

4.4 3.3 Double angle formulae

4.4.1 Key mathematical ideas

The double angle formulae are special cases of the compound angle formulae.

For sine,

\[ \sin2A\equiv2\sin A\cos A. \]

For cosine,

\[ \cos2A\equiv\cos^2A-\sin^2A. \]

Using \(\sin^2A+\cos^2A\equiv1\), this can also be written as

\[ \cos2A\equiv2\cos^2A-1, \]

or

\[ \cos2A\equiv1-2\sin^2A. \]

For tangent,

\[ \tan2A\equiv\dfrac{2\tan A}{1-\tan^2A}. \]

4.4.2 Visualisation using R

Code 
A <- seq(0, 360, length.out = 800)

df <- data.frame(
  A = A
)

df$left_side <- sin_deg(2 * df$A)
df$right_side <- 2 * sin_deg(df$A) * cos_deg(df$A)

  geom_line(aes(y = left_side, linetype = "sin(2A)"), linewidth = 1.1) +
  geom_line(aes(y = right_side, linetype = "2 sin(A) cos(A)"), linewidth = 1.1) +
  geom_hline(yintercept = 0) +

4.4.3 Worked example

If \(\sin A=\dfrac{3}{5}\) and \(A\) is acute, find \(\sin2A\) and \(\cos2A\).

Since \(A\) is acute,

\[ \cos A>0. \]

Using \(\sin^2A+\cos^2A=1\),

\[ \cos A=\dfrac{4}{5}. \]

Therefore,

\[ \sin2A=2\sin A\cos A. \]

So

\[ \sin2A=2\cdot\dfrac{3}{5}\cdot\dfrac{4}{5}=\dfrac{24}{25}. \]

Also,

\[ \cos2A=\cos^2A-\sin^2A. \]

Thus,

\[ \cos2A=\dfrac{16}{25}-\dfrac{9}{25}=\dfrac{7}{25}. \]

4.5 3.4 Further trigonometric identities

4.5.1 Key mathematical ideas

The reciprocal functions give two important identities.

Since

\[ \sec\theta=\dfrac{1}{\cos\theta}, \]

and

\[ 1+\tan^2\theta\equiv\sec^2\theta, \]

we have

\[ \sec^2\theta-\tan^2\theta\equiv1. \]

Similarly,

\[ 1+\cot^2\theta\equiv\operatorname{cosec}^2\theta, \]

so

\[ \operatorname{cosec}^2\theta-\cot^2\theta\equiv1. \]

Many trigonometric identities are proved by converting all expressions to sine and cosine.

4.5.2 Visualisation using R

Code 
theta <- seq(0, 360, length.out = 3000)

  theta = theta
)

df$y <- (1 / cos_deg(df$theta))^2 - tan_deg(df$theta)^2
df$y[abs(cos_deg(df$theta)) < 0.03] <- NA

ggplot(df, aes(x = theta, y = y)) +
  geom_line(linewidth = 1.1, na.rm = TRUE) +
  geom_hline(yintercept = 1, linetype = "dashed") +
  coord_cartesian(ylim = c(-2, 4)) +
  labs(
    title = "Identity: sec^2(theta) - tan^2(theta) = 1",
    subtitle = "Undefined points near asymptotes are omitted.",
    x = "theta in degrees",
    y = "value"
  ) +
  book_theme()

4.5.3 Worked example

Prove that

\[ (\sec x+\tan x)(\sec x-\tan x)\equiv1. \]

Using difference of two squares,

\[ (\sec x+\tan x)(\sec x-\tan x) = \sec^2x-\tan^2x. \]

But

\[ \sec^2x-\tan^2x\equiv1. \]

Therefore,

\[ (\sec x+\tan x)(\sec x-\tan x)\equiv1. \]

4.6 3.5 Expressing \(a\sin\theta+b\cos\theta\) in the form \(R\sin(\theta\pm\alpha)\) or \(R\cos(\theta\pm\alpha)\)

4.6.1 Key mathematical ideas

Expressions of the form

\[ a\sin\theta+b\cos\theta \]

can be written as a single sine or cosine function.

For example,

\[ a\sin\theta+b\cos\theta = R\sin(\theta+\alpha). \]

Expanding the right-hand side gives

\[ R\sin(\theta+\alpha) = R\sin\theta\cos\alpha+R\cos\theta\sin\alpha. \]

Comparing coefficients,

\[ R\cos\alpha=a \]

and

\[ R\sin\alpha=b. \]

Therefore,

\[ R=\sqrt{a^2+b^2} \]

and

\[ \tan\alpha=\dfrac{b}{a}. \]

This form is useful because it makes maximum and minimum values clear.

The maximum value of

\[ R\sin(\theta+\alpha) \]

is \(R\), and the minimum value is \(-R\).

4.6.2 Visualisation using R

Code 
theta <- seq(0, 360, length.out = 800)

alpha <- atan2(4, 3) / deg

df <- data.frame(
  theta = theta
)

df$original <- 3 * sin_deg(df$theta) + 4 * cos_deg(df$theta)
df$single_sine <- 5 * sin_deg(df$theta + alpha)

ggplot(df, aes(x = theta)) +
  geom_line(aes(y = original, linetype = "3sin(theta) + 4cos(theta)"), linewidth = 1.1) +
  geom_line(aes(y = single_sine, linetype = "5sin(theta + alpha)"), linewidth = 1.1) +
  geom_hline(yintercept = 0) +
  labs(
    title = "Combining sine and cosine",
    subtitle = paste0("Here alpha = ", round(alpha, 1), " degrees."),
    x = "theta in degrees",
    y = "value",
    linetype = "Expression"
  ) +
  book_theme()

4.6.3 Worked example

Express

\[ 3\sin\theta+4\cos\theta \]

in the form

\[ R\sin(\theta+\alpha), \]

where \(R>0\) and \(0^\circ<\alpha<90^\circ\).

We write

\[ 3\sin\theta+4\cos\theta = R\sin(\theta+\alpha). \]

Expanding,

\[ R\sin(\theta+\alpha) = R\sin\theta\cos\alpha+R\cos\theta\sin\alpha. \]

Comparing coefficients gives

\[ R\cos\alpha=3 \]

and

\[ R\sin\alpha=4. \]

Squaring and adding,

\[ R^2=3^2+4^2=25. \]

So

\[ R=5. \]

Also,

\[ \tan\alpha=\dfrac{4}{3}. \]

Therefore,

\[ \alpha=\tan^{-1}\left(\dfrac{4}{3}\right). \]

Hence,

\[ 3\sin\theta+4\cos\theta = 5\sin\left(\theta+\tan^{-1}\left(\dfrac{4}{3}\right)\right). \]

## 3.1 The cosecant, secant and cotangent ratios

### Key mathematical ideas

The three reciprocal trigonometric functions are defined by

$$
\operatorname{cosec}\theta=\dfrac{1}{\sin\theta},
$$

$$
\sec\theta=\dfrac{1}{\cos\theta},
$$

and

$$
\cot\theta=\dfrac{1}{\tan\theta}.
$$

Since

$$
\tan\theta=\dfrac{\sin\theta}{\cos\theta},
$$

we also have

$$
\cot\theta=\dfrac{\cos\theta}{\sin\theta}.
$$

These definitions immediately show where the reciprocal functions are undefined.

The function $\operatorname{cosec}\theta$ is undefined when

$$
\sin\theta=0.
$$

The function $\sec\theta$ is undefined when

$$
\cos\theta=0.
$$

The function $\cot\theta$ is undefined when

$$
\sin\theta=0.
$$

### Worked example

Find the exact value of

$$
\operatorname{cosec}240^\circ.
$$

Since

$$
\operatorname{cosec}\theta=\dfrac{1}{\sin\theta},
$$

we have

$$
\operatorname{cosec}240^\circ=\dfrac{1}{\sin240^\circ}.
$$

Now $240^\circ$ is in the third quadrant, where sine is negative. Its reference angle is $60^\circ$, so

$$
\sin240^\circ=-\sin60^\circ.
$$

Therefore,

$$
\sin240^\circ=-\dfrac{\sqrt3}{2}.
$$

Hence,

$$
\operatorname{cosec}240^\circ=-\dfrac{2}{\sqrt3}
=
-\dfrac{2\sqrt3}{3}.
$$

## 3.2 Compound angle formulae

### Key mathematical ideas

The compound angle formulae allow us to expand trigonometric functions of sums and differences.

For sine,

$$
\sin(A+B)\equiv\sin A\cos B+\cos A\sin B,
$$

and

$$
\sin(A-B)\equiv\sin A\cos B-\cos A\sin B.
$$

For cosine,

$$
\cos(A+B)\equiv\cos A\cos B-\sin A\sin B,
$$

and

$$
\cos(A-B)\equiv\cos A\cos B+\sin A\sin B.
$$

For tangent,

$$
\tan(A+B)\equiv
\dfrac{\tan A+\tan B}{1-\tan A\tan B},
$$

and

$$
\tan(A-B)\equiv
\dfrac{\tan A-\tan B}{1+\tan A\tan B}.
$$

Write

\dfrac{\sqrt6+\sqrt2}{4}.
$$

## 3.3 Double angle formulae

### Key mathematical ideas

The double angle formulae are special cases of the compound angle formulae.

For sine,

$$
\sin2A\equiv2\sin A\cos A.
$$

For cosine,

$$
\cos2A\equiv\cos^2A-\sin^2A.
$$

Using $\sin^2A+\cos^2A\equiv1$, this can also be written as

$$
\cos2A\equiv2\cos^2A-1,
$$

or

$$
\cos2A\equiv1-2\sin^2A.
$$

For tangent,

$$
\tan2A\equiv\dfrac{2\tan A}{1-\tan^2A}.
$$

Since $A$ is acute,

$$
\cos A>0.
$$

Using $\sin^2A+\cos^2A=1$,

$$
\cos A=\dfrac{4}{5}.
$$

Therefore,

$$
\sin2A=2\sin A\cos A.
$$

So

$$
\sin2A=2\cdot\dfrac{3}{5}\cdot\dfrac{4}{5}=\dfrac{24}{25}.
$$

Also,

$$
\cos2A=\cos^2A-\sin^2A.
$$

Thus,

$$
\cos2A=\dfrac{16}{25}-\dfrac{9}{25}=\dfrac{7}{25}.
$$

## 3.4 Further trigonometric identities

### Key mathematical ideas

The reciprocal functions give two important identities.

Since

$$
\sec\theta=\dfrac{1}{\cos\theta},
$$

and

$$
1+\tan^2\theta\equiv\sec^2\theta,
$$

we have

$$
\sec^2\theta-\tan^2\theta\equiv1.
$$

Similarly,

$$
1+\cot^2\theta\equiv\operatorname{cosec}^2\theta,
$$

so

$$
\operatorname{cosec}^2\theta-\cot^2\theta\equiv1.
$$

Many trigonometric identities are proved by converting all expressions to sine and cosine.

## 3.5 Expressing $a\sin\theta+b\cos\theta$ in the form $R\sin(\theta\pm\alpha)$ or $R\cos(\theta\pm\alpha)$

### Key mathematical ideas

Expressions of the form

$$
a\sin\theta+b\cos\theta
$$

can be written as a single sine or cosine function.

For example,

$$
a\sin\theta+b\cos\theta
=
R\sin(\theta+\alpha).
$$

Expanding the right-hand side gives

$$
R\sin(\theta+\alpha)
=
R\sin\theta\cos\alpha+R\cos\theta\sin\alpha.
$$

Comparing coefficients,

$$
R\cos\alpha=a
$$

and

$$
R\sin\alpha=b.
$$

Therefore,

$$
R=\sqrt{a^2+b^2}
$$

and

$$
\tan\alpha=\dfrac{b}{a}.
$$

This form is useful because it makes maximum and minimum values clear.

The maximum value of

$$
R\sin(\theta+\alpha)
$$

is $R$, and the minimum value is $-R$.

### Visualisation using R

```{r}
#| label: fig-rsin-form
#| fig-cap: "Writing 3sin(theta) + 4cos(theta) as 5sin(theta + alpha)."
#| fig-width: 6
#| fig-height: 4

alpha <- atan2(4, 3) / deg

df <- data.frame(
  theta = theta
)

df$original <- 3 * sin_deg(df$theta) + 4 * cos_deg(df$theta)
df$single_sine <- 5 * sin_deg(df$theta + alpha)

ggplot(df, aes(x = theta)) +
  geom_line(aes(y = original, linetype = "3sin(theta) + 4cos(theta)"), linewidth = 1.1) +
  geom_line(aes(y = single_sine, linetype = "5sin(theta + alpha)"), linewidth = 1.1) +
  geom_hline(yintercept = 0) +
  labs(
    title = "Combining sine and cosine",
    subtitle = paste0("Here alpha = ", round(alpha, 1), " degrees."),
    x = "theta in degrees",
    y = "value",
    linetype = "Expression"
  ) +
  book_theme()
```

### Worked example

Express

$$
3\sin\theta+4\cos\theta
$$

in the form

$$
R\sin(\theta+\alpha),
$$

where $R>0$ and $0^\circ<\alpha<90^\circ$.

We write

$$
3\sin\theta+4\cos\theta
=
R\sin(\theta+\alpha).
$$

Expanding,

$$
R\sin(\theta+\alpha)
=
R\sin\theta\cos\alpha+R\cos\theta\sin\alpha.
$$

Comparing coefficients gives

$$
R\cos\alpha=3
$$

and

$$
R\sin\alpha=4.
$$

Squaring and adding,

$$
R^2=3^2+4^2=25.
$$

So

$$
R=5.
$$

Also,

$$
\tan\alpha=\dfrac{4}{3}.
$$

Therefore,

$$
\alpha=\tan^{-1}\left(\dfrac{4}{3}\right).
$$

Hence,

$$
3\sin\theta+4\cos\theta
=
5\sin\left(\theta+\tan^{-1}\left(\dfrac{4}{3}\right)\right).
$$
