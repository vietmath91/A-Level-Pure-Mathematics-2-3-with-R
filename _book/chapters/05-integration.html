6.1 Overview

This chapter develops five connected integration ideas:

  1. integration of exponential functions;
  2. integration of expressions of the form \(\dfrac{1}{ax+b}\);
  3. integration of \(\sin(ax+b)\), \(\cos(ax+b)\) and \(\sec^2(ax+b)\);
  4. further integration of trigonometric functions;
  5. the trapezium rule.

The central aim is to treat integration as reverse differentiation, while also developing numerical and graphical ways to estimate areas.

Note

The exercises are organised into tiers.

Bronze develops fluency.
Silver develops simple application.
Gold develops exam-style reasoning.
The STEP-style section extends the chapter beyond routine A Level fluency.

6.2 5.1 Integration of exponential functions

6.2.1 Key mathematical ideas

Since

\[ \dfrac{d}{dx}(e^x)=e^x, \]

we have

\[ \int e^x\,dx=e^x+C. \]

More generally,

\[ \int e^{ax+b}\,dx=\dfrac{1}{a}e^{ax+b}+C, \quad a\neq0. \]

This is because differentiating \(\dfrac{1}{a}e^{ax+b}\) gives

\[ \dfrac{1}{a}\cdot ae^{ax+b}=e^{ax+b}. \]

6.2.2 Visualisation using R

Code 
df <- data.frame(

)

ggplot(df, aes(x = x, y = y)) +
  geom_area(alpha = 0.3) +
  geom_line(linewidth = 1.1) +
  geom_hline(yintercept = 0) +
  labs(
    title = "Area under y = e^x",
    subtitle = "The exact area from 0 to 1 is e - 1.",
    x = "x",
    y = "y"
  ) +
  book_theme()

The exact area is

\[ \int_0^1 e^x\,dx = \left[e^x\right]_0^1 = e-1. \]

6.2.3 Worked example

Find

\[ \int e^{3x-2}\,dx. \]

Since the derivative of \(3x-2\) is \(3\), we divide by \(3\):

\[ \int e^{3x-2}\,dx = \dfrac{1}{3}e^{3x-2}+C. \]

Check by differentiating:

\[ \dfrac{d}{dx}\left(\dfrac{1}{3}e^{3x-2}\right) = \dfrac{1}{3}\cdot3e^{3x-2} = e^{3x-2}. \]

6.3 5.2 Integration of \(\dfrac{1}{ax+b}\)

6.3.1 Key mathematical ideas

Since

\[ \dfrac{d}{dx}\left(\ln|x|\right)=\dfrac{1}{x}, \]

we have

\[ \int \dfrac{1}{x}\,dx=\ln|x|+C. \]

More generally,

\[ \int \dfrac{1}{ax+b}\,dx = \dfrac{1}{a}\ln|ax+b|+C, \quad a\neq0. \]

This follows because differentiating \(\ln|ax+b|\) gives

\[ \dfrac{a}{ax+b}. \]

Therefore, if the numerator is exactly the derivative of the denominator, the integral becomes a logarithm.

  geom_line(linewidth = 1.1) +

    title = "Area under y = 1/(x + 1)",
    subtitle = "The exact area from 0 to 2 is ln 3.",
    x = "x",
    y = "y"

The exact area is

\[ \int_0^2 \dfrac{1}{x+1}\,dx = \left[\ln|x+1|\right]_0^2 = \ln3. \]

6.3.3 Worked example

Find

\[ \int \dfrac{5}{2x-3}\,dx. \]

Since

\[ \int \dfrac{1}{2x-3}\,dx = \dfrac{1}{2}\ln|2x-3|+C, \]

we have

\[ \int \dfrac{5}{2x-3}\,dx = \dfrac{5}{2}\ln|2x-3|+C. \]

6.4 5.3 Integration of \(\sin(ax+b)\), \(\cos(ax+b)\) and \(\sec^2(ax+b)\)

6.4.1 Key mathematical ideas

Angles are measured in radians.

The standard trigonometric integrals are

\[ \int \sin x\,dx=-\cos x+C, \]

\[ \int \cos x\,dx=\sin x+C, \]

and

\[ \int \sec^2x\,dx=\tan x+C. \]

Using the chain rule in reverse,

\[ \int \sin(ax+b)\,dx = -\dfrac{1}{a}\cos(ax+b)+C, \]

\[ \int \cos(ax+b)\,dx = \dfrac{1}{a}\sin(ax+b)+C, \]

and

\[ \int \sec^2(ax+b)\,dx = \dfrac{1}{a}\tan(ax+b)+C. \]

6.4.3 Worked example

Find

\[ \int 6\sin(3x-1)\,dx. \]

Since

\[ \int \sin(3x-1)\,dx = -\dfrac{1}{3}\cos(3x-1)+C, \]

we have

\[ \int 6\sin(3x-1)\,dx = -2\cos(3x-1)+C. \]

Check by differentiating:

\[ \dfrac{d}{dx}\left(-2\cos(3x-1)\right) = 6\sin(3x-1). \]

6.5 5.4 Further integration of trigonometric functions

6.5.1 Key mathematical ideas

Some trigonometric integrals require identities before they can be integrated.

The most useful identities in this section are

\[ \sin^2x=\dfrac{1-\cos2x}{2}, \]

\[ \cos^2x=\dfrac{1+\cos2x}{2}, \]

and

\[ 2\sin x\cos x=\sin2x. \]

These identities turn powers or products of trigonometric functions into expressions that can be integrated directly.

6.5.2 Visualisation using R

Code 
df <- data.frame(
  x = seq(0, 2 * pi, length.out = 800)

  geom_line(aes(y = right_side, linetype = "(1 - cos(2x))/2"), linewidth = 1.1) +
  labs(

    subtitle = "The two graphs overlap.",
    x = "x",
    y = "value",
    linetype = "Expression"
  ) +
  book_theme()

6.5.3 Worked example

Find

\[ \int \sin^2x\,dx. \]

Using the identity

\[ \sin^2x=\dfrac{1-\cos2x}{2}, \]

we get

\[ \int \sin^2x\,dx = \int \dfrac{1-\cos2x}{2}\,dx. \]

So

\[ \int \sin^2x\,dx = \dfrac{x}{2}-\dfrac{1}{4}\sin2x+C. \]

6.6 5.5 The trapezium rule

6.6.1 Key mathematical ideas

The trapezium rule estimates the area under a curve by splitting the interval into equal strips and replacing each curved strip with a trapezium.

Suppose we want to estimate

\[ \int_a^b f(x)\,dx \]

using \(n\) equal strips.

The strip width is

\[ h=\dfrac{b-a}{n}. \]

The \(x\)-values are

\[ x_0=a,\quad x_1=a+h,\quad x_2=a+2h,\quad \ldots,\quad x_n=b. \]

Let

\[ y_i=f(x_i). \]

Then the trapezium rule gives

\[ \int_a^b f(x)\,dx \approx \dfrac{h}{2} \left[ y_0+2y_1+2y_2+\cdots+2y_{n-1}+y_n \right]. \]

6.6.2 Visualisation using R

Code 
f <- function(x) x^2

curve_df <- data.frame(
  x = seq(0, 2, length.out = 800)
)

curve_df$y <- f(curve_df$x)

x_points <- seq(0, 2, length.out = 5)
y_points <- f(x_points)

trapezia <- do.call(
  rbind,
  lapply(1:(length(x_points) - 1), function(i) {
    data.frame(
      strip = i,
      x = c(x_points[i], x_points[i], x_points[i + 1], x_points[i + 1]),
      y = c(0, y_points[i], y_points[i + 1], 0)
    )
  })
)

ggplot() +
  geom_polygon(
    data = trapezia,
    aes(x = x, y = y, group = strip),
    alpha = 0.3
  ) +
  geom_line(
    data = curve_df,
    aes(x = x, y = y),
    linewidth = 1.1
  ) +
  geom_point(
    data = data.frame(x = x_points, y = y_points),
    aes(x = x, y = y),
    size = 2
  ) +
  geom_hline(yintercept = 0) +
  labs(
    title = "Trapezium rule",
    subtitle = "The curve y = x^2 is approximated by straight-line segments.",
    x = "x",
    y = "y"
  ) +
  book_theme()

6.6.3 Worked example

Use the trapezium rule with \(4\) strips to estimate

\[ \int_0^2 x^2\,dx. \]

Here,

\[ a=0,\quad b=2,\quad n=4. \]

Therefore,

\[ h=\dfrac{2-0}{4}=\dfrac{1}{2}. \]

The \(x\)-values are

\[ 0,\quad \dfrac{1}{2},\quad 1,\quad \dfrac{3}{2},\quad 2. \]

The corresponding \(y\)-values for \(y=x^2\) are

\[ 0,\quad \dfrac{1}{4},\quad 1,\quad \dfrac{9}{4},\quad 4. \]

Using the trapezium rule,

\[ T = \dfrac{h}{2} \left[ y_0+2y_1+2y_2+2y_3+y_4 \right]. \]

So

\[ T = \dfrac{1}{4} \left[ 0+2\left(\dfrac{1}{4}\right)+2(1)+2\left(\dfrac{9}{4}\right)+4 \right]. \]

Therefore,

\[ T=\dfrac{11}{4}. \]

6.6.4 Short investigation

Change the number of strips and observe the estimate.

Code 
trapezium_estimate <- function(f, a, b, n) {
  x <- seq(a, b, length.out = n + 1)
  y <- f(x)
  h <- (b - a) / n
  (h / 2) * (y[1] + 2 * sum(y[2:n]) + y[n + 1])
}

n_values <- 1:20

df <- data.frame(
  n = n_values,
  estimate = sapply(
    n_values,
    function(n) trapezium_estimate(function(x) x^2, 0, 2, n)
  )
)

df$exact <- 8 / 3

ggplot(df, aes(x = n)) +
  geom_line(aes(y = estimate, linetype = "trapezium estimate"), linewidth = 1.1) +
  geom_line(aes(y = exact, linetype = "exact value"), linewidth = 1.1) +
  labs(
    title = "Trapezium rule estimates",
    subtitle = "As the number of strips increases, the estimate approaches the exact value.",
    x = "number of strips",
    y = "area estimate",
    linetype = "Quantity"
  ) +
  book_theme()

## 5.1 Integration of exponential functions

### Key mathematical ideas

Since

$$
\dfrac{d}{dx}(e^x)=e^x,
$$

we have

$$
\int e^x\,dx=e^x+C.
$$

More generally,

$$
\int e^{ax+b}\,dx=\dfrac{1}{a}e^{ax+b}+C,
\quad a\neq0.
$$

This is because differentiating $\dfrac{1}{a}e^{ax+b}$ gives

$$
\dfrac{1}{a}\cdot ae^{ax+b}=e^{ax+b}.
$$

### Visualisation using R

```{r}

#| fig-height: 4

  geom_line(linewidth = 1.1) +

    subtitle = "The exact area from 0 to 1 is e - 1.",
    x = "x",
    y = "y"
  ) +
  book_theme()
```

The exact area is

$$
\int_0^1 e^x\,dx
=
\left[e^x\right]_0^1
=
e-1.
$$

### Worked example

Find

$$
\int e^{3x-2}\,dx.
$$

Since the derivative of $3x-2$ is $3$, we divide by $3$:

$$
\int e^{3x-2}\,dx
=
\dfrac{1}{3}e^{3x-2}+C.
$$

Check by differentiating:

$$
\dfrac{d}{dx}\left(\dfrac{1}{3}e^{3x-2}\right)
=
\dfrac{1}{3}\cdot3e^{3x-2}
=
e^{3x-2}.
$$

## 5.2 Integration of $\dfrac{1}{ax+b}$

### Key mathematical ideas

Since

$$
\dfrac{d}{dx}\left(\ln|x|\right)=\dfrac{1}{x},
$$

we have

$$
\int \dfrac{1}{x}\,dx=\ln|x|+C.
$$

More generally,

$$
\int \dfrac{1}{ax+b}\,dx
=
\dfrac{1}{a}\ln|ax+b|+C,
\quad a\neq0.
$$

This follows because differentiating $\ln|ax+b|$ gives

$$
\dfrac{a}{ax+b}.
$$

Therefore, if the numerator is exactly the derivative of the denominator, the integral becomes a logarithm.

## 5.3 Integration of $\sin(ax+b)$, $\cos(ax+b)$ and $\sec^2(ax+b)$

### Key mathematical ideas

Angles are measured in radians.

The standard trigonometric integrals are

$$
\int \sin x\,dx=-\cos x+C,
$$

$$
\int \cos x\,dx=\sin x+C,
$$

and

$$
\int \sec^2x\,dx=\tan x+C.
$$

Using the chain rule in reverse,

$$
\int \sin(ax+b)\,dx
=
-\dfrac{1}{a}\cos(ax+b)+C,
$$

$$
\int \cos(ax+b)\,dx
=
\dfrac{1}{a}\sin(ax+b)+C,
$$

and

$$
\int \sec^2(ax+b)\,dx
=
\dfrac{1}{a}\tan(ax+b)+C.
$$

## 5.4 Further integration of trigonometric functions

### Key mathematical ideas

Some trigonometric integrals require identities before they can be integrated.

The most useful identities in this section are

$$
\sin^2x=\dfrac{1-\cos2x}{2},
$$

$$
\cos^2x=\dfrac{1+\cos2x}{2},
$$

and

$$
2\sin x\cos x=\sin2x.
$$

These identities turn powers or products of trigonometric functions into expressions that can be integrated directly.

## 5.5 The trapezium rule

### Key mathematical ideas

The trapezium rule estimates the area under a curve by splitting the interval into equal strips and replacing each curved strip with a trapezium.

Suppose we want to estimate

$$
\int_a^b f(x)\,dx
$$

using $n$ equal strips.

The strip width is

$$
h=\dfrac{b-a}{n}.
$$

The $x$-values are

$$
x_0=a,\quad x_1=a+h,\quad x_2=a+2h,\quad \ldots,\quad x_n=b.
$$

Let

$$
y_i=f(x_i).
$$

Then the trapezium rule gives

$$
\int_a^b f(x)\,dx
\approx
\dfrac{h}{2}
\left[
y_0+2y_1+2y_2+\cdots+2y_{n-1}+y_n
\right].
$$

### Worked example

Use the trapezium rule with $4$ strips to estimate

$$
\int_0^2 x^2\,dx.
$$

Here,

$$
a=0,\quad b=2,\quad n=4.
$$

Therefore,

$$
h=\dfrac{2-0}{4}=\dfrac{1}{2}.
$$

The $x$-values are

$$
0,\quad \dfrac{1}{2},\quad 1,\quad \dfrac{3}{2},\quad 2.
$$

The corresponding $y$-values for $y=x^2$ are

$$
0,\quad \dfrac{1}{4},\quad 1,\quad \dfrac{9}{4},\quad 4.
$$

Using the trapezium rule,

$$
T
=
\dfrac{h}{2}
\left[
y_0+2y_1+2y_2+2y_3+y_4
\right].
$$

So

$$
T
=
\dfrac{1}{4}
\left[
0+2\left(\dfrac{1}{4}\right)+2(1)+2\left(\dfrac{9}{4}\right)+4
\right].
$$

Therefore,

$$
T=\dfrac{11}{4}.
$$

### Short investigation

Change the number of strips and observe the estimate.

```{r}
#| label: fig-trapezium-estimate-changing-n
#| fig-cap: "Trapezium rule estimates for y = x^2 on [0, 2]."
#| fig-width: 6
#| fig-height: 4

trapezium_estimate <- function(f, a, b, n) {
  x <- seq(a, b, length.out = n + 1)
  y <- f(x)
  h <- (b - a) / n
  (h / 2) * (y[1] + 2 * sum(y[2:n]) + y[n + 1])
}

n_values <- 1:20

df <- data.frame(
  n = n_values,
  estimate = sapply(
    n_values,
    function(n) trapezium_estimate(function(x) x^2, 0, 2, n)
  )
)
```

### Answers to STEP-style extension questions

1. Since
   $$
   \sin^2x=\dfrac{1-\cos2x}{2},
   $$
   we have
   $$
   \int_0^\pi \sin^2x\,dx
   =
   \int_0^\pi \dfrac{1-\cos2x}{2}\,dx.
   $$
   Therefore,
   $$
   \int_0^\pi \sin^2x\,dx
   =
   \left[\dfrac{x}{2}-\dfrac{1}{4}\sin2x\right]_0^\pi
   =
   \dfrac{\pi}{2}.
   $$

2. Since $e^{ka}=4$,
   $$
   \int_0^a e^{kx}\,dx
   =
   \left[\dfrac{1}{k}e^{kx}\right]_0^a
   =
   \dfrac{4-1}{k}.
   $$
   This is equal to $3$, so $k=1$. Hence $e^a=4$, giving $a=\ln4$.

3. Define
   $$
   g(a)=a-\ln(1+a).
   $$
   Then
   $$
   g'(a)=1-\dfrac{1}{1+a}=\dfrac{a}{1+a}.
   $$
   For $a>0$, $g'(a)>0$, and $g(0)=0$. Therefore $g(a)>0$ for $a>0$, so
   $$
   \ln(1+a)<a.
   $$
   Since
   $$
   \ln(1+a)=\int_0^a \dfrac{1}{1+x}\,dx,
   $$
   this says that the area under $y=\dfrac{1}{1+x}$ from $0$ to $a$ is less than the area of the rectangle of width $a$ and height $1$.

4. We require
   $$
   F'(x)=6e^{2x}+12\cos3x.
   $$
   Since
   $$
   F'(x)=2Ae^{2x}+3B\cos3x,
   $$
   we get
   $$
   2A=6
   $$
   and
   $$
   3B=12.
   $$
   Therefore,
   $$
   A=3
   \quad \text{and} \quad
   B=4.
   $$

5. The strip width is
   $$
   h=\dfrac{a}{n}.
   $$
   The ordinates are
   $$
   y_i=\left(\dfrac{ia}{n}\right)^2.
   $$
   The trapezium estimate is
   $$
   T=
   \dfrac{h}{2}
   \left[
   y_0+2y_1+\cdots+2y_{n-1}+y_n
   \right].
   $$
   Since $y_0=0$ and $y_n=a^2$,
   $$
   T
   =
   \dfrac{a}{2n}
   \left[
   2\sum_{i=1}^{n-1}\left(\dfrac{ia}{n}\right)^2+a^2
   \right].
   $$
   Hence
   $$
   T=
   \dfrac{a^3}{2n}
   \left[
   \dfrac{2}{n^2}\sum_{i=1}^{n-1}i^2+1
   \right].
   $$
   Using
   $$
   \sum_{i=1}^{n-1}i^2=\dfrac{(n-1)n(2n-1)}{6},
   $$
   we obtain
   $$
   T=\dfrac{a^3}{3}+\dfrac{a^3}{6n^2}.
   $$
   But
   $$
   \int_0^a x^2\,dx=\dfrac{a^3}{3}.
   $$
   Therefore the trapezium rule overestimates the exact value by
   $$
   \dfrac{a^3}{6n^2}.
   $$
