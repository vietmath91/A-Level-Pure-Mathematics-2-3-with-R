3.1 Overview

This chapter develops eight connected ideas:

  1. logarithms to base \(10\);
  2. logarithms to base \(a\);
  3. the laws of logarithms;
  4. solving logarithmic equations;
  5. solving exponential equations;
  6. solving exponential inequalities;
  7. natural logarithms;
  8. transforming relationships to linear form.

The main theme is that logarithms and exponentials are inverse processes. Logarithms allow us to solve equations where the unknown appears in an index, and they also allow us to convert some curved relationships into straight-line relationships.

3.2 2.1 Logarithms to base 10

3.2.1 Key mathematical ideas

A logarithm answers the question:

What power is needed?

For example,

\[ 10^2=100. \]

So

\[ \log_{10}100=2. \]

In general,

\[ \log_{10}x=y \quad \Longleftrightarrow \quad 10^y=x. \]

The expression \(\log_{10}x\) is defined only when

\[ x>0. \]

3.2.2 Visualisation using R

Code 
df <- data.frame(
  x = seq(0.05, 100, length.out = 800)
)

df$y <- log10(df$x)

ggplot(df, aes(x = x, y = y)) +
  geom_line(linewidth = 1.1) +
  geom_hline(yintercept = 0) +
  geom_vline(xintercept = 0) +
  geom_vline(xintercept = 1, linetype = "dashed") +
  geom_vline(xintercept = 10, linetype = "dashed") +
  geom_hline(yintercept = 1, linetype = "dashed") +
  labs(
    title = "Graph of y = log10(x)",
    subtitle = "The graph passes through (1, 0) and (10, 1).",
    x = "x",
    y = "log10(x)"
  ) +
  book_theme()

The graph passes through \((1,0)\) because

\[ \log_{10}1=0. \]

It passes through \((10,1)\) because

\[ \log_{10}10=1. \]

3.2.3 Worked example

Evaluate

\[ \log_{10}1000. \]

Since

\[ 1000=10^3, \]

we have

\[ \log_{10}1000=3. \]

3.3 2.2 Logarithms to base \(a\)

3.3.1 Key mathematical ideas

For \(a>0\) and \(a\neq1\),

\[ \log_a x=y \quad \Longleftrightarrow \quad a^y=x. \]

For example,

\[ 2^3=8 \]

so

\[ \log_2 8=3. \]

Important facts include:

\[ \log_a1=0, \]

\[ \log_a a=1, \]

and

\[ \log_a(a^x)=x. \]

3.4 2.3 The laws of logarithms

3.4.1 Key mathematical ideas

For \(a>0\), \(a\neq1\), and positive \(x\) and \(y\),

\[ \log_a(xy)=\log_a x+\log_a y. \]

This is the product rule.

\[ \log_a\left(\frac{x}{y}\right)=\log_a x-\log_a y. \]

This is the division rule.

\[ \log_a(x^n)=n\log_a x. \]

This is the power rule.

A useful special case is

\[ \log_a\left(\frac{1}{x}\right)=-\log_a x. \]

3.4.2 Visualisation using R

Code

)

The two graphs overlap because the two expressions are equivalent.

3.4.3 Worked example

Simplify

\[ \log_a(3x^2)-\log_a(6x). \]

Using the division rule,

\[ \log_a(3x^2)-\log_a(6x) = \log_a\left(\frac{3x^2}{6x}\right). \]

So

\[ \log_a(3x^2)-\log_a(6x) = \log_a\left(\frac{x}{2}\right), \]

where \(x>0\).

3.5 2.4 Solving logarithmic equations

3.5.1 Key mathematical ideas

When solving logarithmic equations, always check the domain.

For example,

\[ \log_a(x-3) \]

is defined only when

\[ x-3>0. \]

So

\[ x>3. \]

If

\[ \log_a A=\log_a B, \]

then

\[ A=B, \]

provided \(A>0\) and \(B>0\).

3.5.2 Worked example

Solve

\[ \log_2(x+6)-\log_2x=2. \]

First, the domain requires

\[ x>0. \]

Using the division rule,

\[ \log_2\left(\frac{x+6}{x}\right)=2. \]

Therefore,

\[ \frac{x+6}{x}=2^2. \]

So

\[ \frac{x+6}{x}=4. \]

Hence,

\[ x+6=4x. \]

Therefore,

\[ 3x=6, \]

so

\[ x=2. \]

Since \(x=2>0\), the solution is valid.

3.6 2.5 Solving exponential equations

3.6.1 Key mathematical ideas

Exponential equations often have the unknown in an index.

For example,

\[ 2^x=7. \]

Taking natural logarithms gives

\[ \ln(2^x)=\ln7. \]

Using the power rule,

\[ x\ln2=\ln7. \]

Therefore,

\[ x=\frac{\ln7}{\ln2}. \]

3.6.2 Visualisation using R

Code 
df <- data.frame(
  x = seq(-2, 5, length.out = 800)
)

df$y <- 2^df$x

solution <- log(7) / log(2)

ggplot(df, aes(x = x, y = y)) +
  geom_line(linewidth = 1.1) +
  geom_hline(yintercept = 7, linetype = "dashed") +
  geom_vline(xintercept = solution, linetype = "dashed") +
  annotate("point", x = solution, y = 7, size = 3) +
  labs(
    title = "Graph of y = 2^x",
    subtitle = "The equation 2^x = 7 has one real solution.",
    x = "x",
    y = "y"
  ) +
  book_theme()

3.6.3 Worked example

Solve

\[ 5^{x-2}=17. \]

Taking natural logarithms,

\[ \ln(5^{x-2})=\ln17. \]

Using the power rule,

\[ (x-2)\ln5=\ln17. \]

Therefore,

\[ x-2=\frac{\ln17}{\ln5}. \]

So

\[ x=2+\frac{\ln17}{\ln5}. \]

3.7 2.6 Solving exponential inequalities

3.7.1 Key mathematical ideas

For \(a>1\), the function

\[ y=a^x \]

is increasing.

Therefore,

\[ a^x>a^k \quad \Longleftrightarrow \quad x>k. \]

For \(0<a<1\), the function

\[ y=a^x \]

is decreasing.

Therefore,

\[ a^x>a^k \quad \Longleftrightarrow \quad x<k. \]

The direction of the inequality depends on whether the base is greater than \(1\) or between \(0\) and \(1\).

Code

  ) +

3.7.3 Worked example

Solve

\[ 3^{2x-1}\geq 27. \]

Since

\[ 27=3^3, \]

we have

\[ 3^{2x-1}\geq3^3. \]

The base \(3\) is greater than \(1\), so the inequality direction is unchanged:

\[ 2x-1\geq3. \]

Therefore,

\[ 2x\geq4, \]

so

\[ x\geq2. \]

3.8 2.7 Natural logarithms

3.8.1 Key mathematical ideas

The number \(e\) is approximately

\[ e\approx2.718. \]

The logarithm to base \(e\) is called the natural logarithm.

We write

\[ \ln x \]

instead of

\[ \log_e x. \]

The functions

\[ y=e^x \]

and

\[ y=\ln x \]

are inverse functions.

This means:

\[ e^{\ln x}=x \]

and

\[ \ln(e^x)=x. \]

3.8.2 Visualisation using R

3.9 2.8 Transforming relationships to linear form

3.9.1 Key mathematical ideas

Logarithms can transform some curved relationships into straight-line relationships.

If

\[ y=kx^n, \]

then taking natural logarithms gives

\[ \ln y=\ln(kx^n). \]

So

\[ \ln y=\ln k+n\ln x. \]

This has the form

\[ Y=mX+c, \]

where

\[ Y=\ln y, \quad X=\ln x, \quad m=n, \quad c=\ln k. \]

If

\[ y=ka^x, \]

then

\[ \ln y=\ln(ka^x). \]

So

\[ \ln y=\ln k+x\ln a. \]

This is linear in \(x\).

3.9.3 Worked example

The variables \(x\) and \(y\) satisfy

\[ y=kx^n. \]

The graph of \(\ln y\) against \(\ln x\) is a straight line passing through \((0,1)\) and \((2,5)\).

Find \(k\) and \(n\).

Since

\[ \ln y=n\ln x+\ln k, \]

the gradient is \(n\).

Therefore,

\[ n=\frac{5-1}{2-0}=2. \]

The intercept is \(1\), so

\[ \ln k=1. \]

Hence,

\[ k=e. \]

Therefore,

\[ y=ex^2. \]

> What power is needed?

For example,

$$
10^2=100.
$$

So

$$
\log_{10}100=2.
$$

In general,

$$
\log_{10}x=y
\quad \Longleftrightarrow \quad
10^y=x.
$$

The expression $\log_{10}x$ is defined only when

## 2.3 The laws of logarithms

### Key mathematical ideas

For $a>0$, $a\neq1$, and positive $x$ and $y$,

$$
\log_a(xy)=\log_a x+\log_a y.
$$

This is the product rule.

$$
\log_a\left(\frac{x}{y}\right)=\log_a x-\log_a y.
$$

This is the division rule.

$$
\log_a(x^n)=n\log_a x.
$$

This is the power rule.

A useful special case is

$$
\log_a\left(\frac{1}{x}\right)=-\log_a x.
$$

### Visualisation using R

## 2.4 Solving logarithmic equations

### Key mathematical ideas

When solving logarithmic equations, always check the domain.

For example,

$$
\log_a(x-3)
$$

is defined only when

$$
x-3>0.
$$

So

$$
x>3.
$$

If

$$
\log_a A=\log_a B,
$$

then

$$
A=B,
$$

provided $A>0$ and $B>0$.

### Worked example

Solve

$$
\log_2(x+6)-\log_2x=2.
$$

First, the domain requires

$$
x>0.
$$

Using the division rule,

$$
\log_2\left(\frac{x+6}{x}\right)=2.
$$

Therefore,

$$
\frac{x+6}{x}=2^2.
$$

So

$$
\frac{x+6}{x}=4.
$$

Hence,

$$
x+6=4x.
$$

Therefore,

$$
3x=6,
$$

so

$$
x=2.
$$

Since $x=2>0$, the solution is valid.

## 2.5 Solving exponential equations

### Key mathematical ideas

Exponential equations often have the unknown in an index.

For example,

$$
2^x=7.
$$

Taking natural logarithms gives

$$
\ln(2^x)=\ln7.
$$

Using the power rule,

$$
x\ln2=\ln7.
$$

Therefore,

$$
x=\frac{\ln7}{\ln2}.
$$

## 2.6 Solving exponential inequalities

### Key mathematical ideas

For $a>1$, the function

$$
y=a^x
$$

is increasing.

Therefore,

$$
a^x>a^k
\quad \Longleftrightarrow \quad
x>k.
$$

For $0<a<1$, the function

$$
y=a^x
$$

is decreasing.

Therefore,

$$
a^x>a^k
\quad \Longleftrightarrow \quad
x<k.
$$

The direction of the inequality depends on whether the base is greater than $1$ or between $0$ and $1$.

## 2.7 Natural logarithms

### Key mathematical ideas

The number $e$ is approximately

$$
e\approx2.718.
$$

The logarithm to base $e$ is called the natural logarithm.

We write

$$
\ln x
$$

instead of

$$
\log_e x.
$$

The functions

$$
y=e^x
$$

and

$$
y=\ln x
$$

are inverse functions.

This means:

$$
e^{\ln x}=x
$$

and

$$
\ln(e^x)=x.
$$

## 2.8 Transforming relationships to linear form

### Key mathematical ideas

Logarithms can transform some curved relationships into straight-line relationships.

If

$$
y=kx^n,
$$

then taking natural logarithms gives

$$
\ln y=\ln(kx^n).
$$

So

$$
\ln y=\ln k+n\ln x.
$$

This has the form

$$
Y=mX+c,
$$

where

$$
Y=\ln y,
\quad
X=\ln x,
\quad
m=n,
\quad
c=\ln k.
$$

If

$$
y=ka^x,
$$

then

$$
\ln y=\ln(ka^x).
$$

So

$$
\ln y=\ln k+x\ln a.
$$

This is linear in $x$.

### Visualisation using R

```{r}
#| label: fig-linearisation-power-law
#| fig-cap: "Linearising y = 3x^2 using logarithms."
#| fig-width: 6
#| fig-height: 4

  x = seq(1, 20, length.out = 200)
)

df$y <- 3 * df$x^2
df$log_x <- log(df$x)
df$log_y <- log(df$y)

ggplot(df, aes(x = log_x, y = log_y)) +
  geom_line(linewidth = 1.1) +
  labs(
    title = "Linear form of y = 3x^2",
    subtitle = "Plotting ln(y) against ln(x) gives a straight line.",
    x = "ln(x)",
    y = "ln(y)"
  ) +
  book_theme()
```

For

$$
y=3x^2,
$$

we have

$$
\ln y=2\ln x+\ln3.
$$

So the gradient of the line is $2$, and the intercept is $\ln3$.

### Worked example

The variables $x$ and $y$ satisfy

$$
y=kx^n.
$$

The graph of $\ln y$ against $\ln x$ is a straight line passing through $(0,1)$ and $(2,5)$.

Find $k$ and $n$.

Since

$$
\ln y=n\ln x+\ln k,
$$

the gradient is $n$.

Therefore,

$$
n=\frac{5-1}{2-0}=2.
$$

The intercept is $1$, so

$$
\ln k=1.
$$

Hence,

$$
k=e.
$$

Therefore,

$$
y=ex^2.
$$

3. Taking natural logarithms gives $\ln y=\ln k+n\ln x$. This is a straight line with vertical variable $\ln y$, horizontal variable $\ln x$, and gradient $n$.
4. Writing each logarithm using natural logarithms gives
   $$
   \frac{\ln b}{\ln a}\cdot\frac{\ln c}{\ln b}\cdot\frac{\ln a}{\ln c}=1.
   $$
5. The domain is $0<x<4$. Combining logarithms gives $\ln(x(4-x))=\ln3$, so $x(4-x)=3$. Hence $x^2-4x+3=0$, giving $x=1$ or $x=3$.
