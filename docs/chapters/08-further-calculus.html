9.1 Overview

This chapter develops seven connected calculus ideas:

  1. the derivative of \(\tan^{-1}x\);
  2. integration of expressions of the form \(\dfrac{1}{x^2+a^2}\);
  3. integration of expressions of the form \(\dfrac{kf'(x)}{f(x)}\);
  4. integration by substitution;
  5. the use of partial fractions in integration;
  6. integration by parts;
  7. further integration, including areas and volumes of revolution.

The central aim is to recognise structure before choosing a method. In this chapter, the difficulty is often not the calculation itself, but deciding which method is appropriate.

Note

The exercises are organised into tiers.

Bronze develops fluency.
Silver develops simple application.
Gold develops exam-style reasoning.
The STEP-style section extends the chapter beyond routine A Level fluency.

9.2 8.1 Derivative of \(\tan^{-1}x\)

9.2.1 Key mathematical ideas

The derivative of the inverse tangent function is

\[ \dfrac{d}{dx}\left(\tan^{-1}x\right) = \dfrac{1}{1+x^2}. \]

Using the chain rule,

\[ \dfrac{d}{dx}\left(\tan^{-1}(f(x))\right) = \dfrac{f'(x)}{1+[f(x)]^2}. \]

In particular,

\[ \dfrac{d}{dx}\left(\tan^{-1}(ax+b)\right) = \dfrac{a}{1+(ax+b)^2}. \]

9.2.2 Visualisation using R

Code 
df <- data.frame(
  x = seq(-6, 6, length.out = 1000)
)

df$inverse_tangent <- atan(df$x)
df$derivative <- 1 / (1 + df$x^2)

ggplot(df, aes(x = x)) +
  geom_line(aes(y = inverse_tangent, linetype = "tan^{-1}(x)"), linewidth = 1.1) +
  geom_line(aes(y = derivative, linetype = "1/(1 + x^2)"), linewidth = 1.1) +
  geom_hline(yintercept = 0) +
  geom_vline(xintercept = 0) +
  labs(
    title = "Inverse tangent and its derivative",
    subtitle = "The derivative is always positive and becomes small as |x| becomes large.",
    x = "x",
    y = "value",
    linetype = "Curve"
  ) +
  book_theme()

This graph is useful because it compares the function with its gradient function.

The curve

\[ y=\tan^{-1}x \]

is increasing, but it gradually flattens out. The derivative

\[ y=\dfrac{1}{1+x^2} \]

explains this: the gradient is positive for all \(x\), but tends to \(0\) as \(|x|\) becomes large.

9.2.3 Worked example

Differentiate

\[ y=\tan^{-1}(3x-2). \]

Let

\[ u=3x-2. \]

Then

\[ \dfrac{du}{dx}=3. \]

Using the chain rule,

\[ \dfrac{dy}{dx} = \dfrac{3}{1+(3x-2)^2}. \]

9.3 8.2 Integration of \(\dfrac{1}{x^2+a^2}\)

9.3.1 Key mathematical ideas

Since

\[ \dfrac{d}{dx}\left(\tan^{-1}x\right)=\dfrac{1}{1+x^2}, \]

we have

\[ \int \dfrac{1}{1+x^2}\,dx=\tan^{-1}x+C. \]

More generally,

\[ \int \dfrac{1}{x^2+a^2}\,dx = \dfrac{1}{a}\tan^{-1}\left(\dfrac{x}{a}\right)+C, \quad a>0. \]

This result is particularly useful when a quadratic denominator cannot be factorised over the real numbers.

9.3.2 Visualisation using R

Code 
df <- data.frame(
  x = seq(0, 1, length.out = 600)
)

df$y <- 1 / (1 + df$x^2)

ggplot(df, aes(x = x, y = y)) +
  geom_area(alpha = 0.3) +
  geom_line(linewidth = 1.1) +
  geom_hline(yintercept = 0) +
  labs(
    title = "Area under y = 1/(1 + x^2)",
    subtitle = "The exact area from 0 to 1 is pi/4.",
    x = "x",
    y = "y"
  ) +
  book_theme()

The exact area is

\[ \int_0^1 \dfrac{1}{1+x^2}\,dx = \left[\tan^{-1}x\right]_0^1 = \dfrac{\pi}{4}. \]

9.3.3 Worked example

Find

\[ \int \dfrac{1}{x^2+9}\,dx. \]

Since

\[ x^2+9=x^2+3^2, \]

we use \(a=3\).

Therefore,

\[ \int \dfrac{1}{x^2+9}\,dx = \dfrac{1}{3}\tan^{-1}\left(\dfrac{x}{3}\right)+C. \]

9.4 8.3 Integration of \(\dfrac{kf'(x)}{f(x)}\)

9.4.1 Key mathematical ideas

If \(f(x)\) is differentiable, then

\[ \dfrac{d}{dx}\left(\ln|f(x)|\right) = \dfrac{f'(x)}{f(x)}. \]

Therefore,

\[ \int \dfrac{f'(x)}{f(x)}\,dx = \ln|f(x)|+C. \]

More generally,

\[ \int \dfrac{kf'(x)}{f(x)}\,dx = k\ln|f(x)|+C. \]

The key is to recognise that the numerator is a constant multiple of the derivative of the denominator.

9.4.2 Visualisation using R

Code 
df <- data.frame(
  x = seq(-3, 3, length.out = 800)
)

df$antiderivative <- log(df$x^2 + 1)

ggplot(df, aes(x = x)) +
  geom_line(aes(y = integrand, linetype = "integrand"), linewidth = 1.1) +
  geom_line(aes(y = antiderivative, linetype = "antiderivative"), linewidth = 1.1) +
  geom_hline(yintercept = 0) +
  labs(
    title = "Recognising f'(x)/f(x)",
    subtitle = "The antiderivative is logarithmic.",
    x = "x",
    y = "value",
    linetype = "Curve"
  ) +
  book_theme()

9.4.3 Worked example

Find

\[ \int \dfrac{6x+4}{3x^2+4x+1}\,dx. \]

Let

\[ f(x)=3x^2+4x+1. \]

Then

\[ f'(x)=6x+4. \]

Therefore,

\[ \int \dfrac{6x+4}{3x^2+4x+1}\,dx = \ln|3x^2+4x+1|+C. \]

9.4.4 Short investigation

Use R to check by differentiating numerically.

9.5 8.4 Integration by substitution

9.5.1 Key mathematical ideas

Integration by substitution is the reverse of the chain rule.

If

\[ u=f(x), \]

then

\[ \dfrac{du}{dx}=f'(x), \]

so

\[ du=f'(x)\,dx. \]

The purpose of substitution is to rewrite the whole integral in terms of a new variable.

A typical pattern is

\[ \int f'(x)[f(x)]^n\,dx. \]

Using

\[ u=f(x), \]

this becomes

\[ \int u^n\,du. \]

9.5.2 Visualisation using R

Code 
df <- data.frame(
  x = seq(0, 1, length.out = 600)
)

df$y <- 2 * df$x * (1 + df$x^2)^2

ggplot(df, aes(x = x, y = y)) +
  geom_area(alpha = 0.3) +
  geom_line(linewidth = 1.1) +
  geom_hline(yintercept = 0) +
  labs(
    title = "A substitution structure",
    subtitle = "For 2x(1 + x^2)^2, use u = 1 + x^2.",
    x = "x",
    y = "y"
  ) +
  book_theme()

9.5.3 Worked example

Find

\[ \int 2x(1+x^2)^4\,dx. \]

Let

\[ u=1+x^2. \]

Then

\[ \dfrac{du}{dx}=2x, \]

so

\[ du=2x\,dx. \]

Therefore,

\[ \int 2x(1+x^2)^4\,dx = \int u^4\,du. \]

Hence,

\[ \int u^4\,du=\dfrac{u^5}{5}+C. \]

Substituting back gives

\[ \int 2x(1+x^2)^4\,dx = \dfrac{(1+x^2)^5}{5}+C. \]

9.6 8.5 The use of partial fractions in integration

9.6.1 Key mathematical ideas

Partial fractions are useful in integration because they split a complicated rational function into simpler terms.

For example,

\[ \dfrac{1}{x(x+1)} = \dfrac{1}{x}-\dfrac{1}{x+1}. \]

Therefore,

\[ \int \dfrac{1}{x(x+1)}\,dx = \ln|x|-\ln|x+1|+C. \]

Warning

Before using partial fractions, check whether the algebraic fraction is proper. If the degree of the numerator is greater than or equal to the degree of the denominator, do long division first.

9.6.2 Visualisation using R

Code 
df <- data.frame(
  x = seq(0.2, 5, length.out = 800)
)

df$original <- 1 / (df$x * (df$x + 1))
df$partial_fraction <- 1 / df$x - 1 / (df$x + 1)

ggplot(df, aes(x = x)) +
  geom_line(aes(y = original, linetype = "original"), linewidth = 1.1) +
  geom_line(aes(y = partial_fraction, linetype = "partial fractions"), linewidth = 1.1) +
  geom_hline(yintercept = 0) +
  labs(
    title = "Partial fractions before integration",
    subtitle = "The two expressions are identical on their common domain.",
    x = "x",
    y = "value",
    linetype = "Expression"
  ) +

9.6.3 Worked example

Find

\[ \int \dfrac{3x+5}{(x+1)(x+2)}\,dx. \]

First write

\[ \dfrac{3x+5}{(x+1)(x+2)} = \dfrac{A}{x+1}+\dfrac{B}{x+2}. \]

Multiplying throughout by \((x+1)(x+2)\) gives

\[ 3x+5=A(x+2)+B(x+1). \]

So

\[ 3x+5=(A+B)x+(2A+B). \]

Equating coefficients gives

\[ A+B=3 \]

and

\[ 2A+B=5. \]

Solving gives

\[ A=2, \quad B=1. \]

Therefore,

\[ \int \dfrac{3x+5}{(x+1)(x+2)}\,dx = \int \left(\dfrac{2}{x+1}+\dfrac{1}{x+2}\right)\,dx. \]

Hence,

\[ \int \dfrac{3x+5}{(x+1)(x+2)}\,dx = 2\ln|x+1|+\ln|x+2|+C. \]

9.7 8.6 Integration by parts

9.7.1 Key mathematical ideas

Integration by parts is the reverse of the product rule.

The product rule says

\[ \dfrac{d}{dx}(uv)=u\dfrac{dv}{dx}+v\dfrac{du}{dx}. \]

Rearranging and integrating gives

\[ \int u\dfrac{dv}{dx}\,dx = uv-\int v\dfrac{du}{dx}\,dx. \]

This is usually written as

\[ \int u\,dv=uv-\int v\,du. \]

9.7.2 Why display integration by parts?

A graph does not show the algebraic mechanism of integration by parts as directly as it shows area or volume. Its value is different: it shows the integral as an area under a product curve, while the method rewrites that area in terms of a boundary term minus a simpler area.

So integration by parts is worth displaying when we want students to see that an apparently difficult product integral such as

\[ \int xe^x\,dx \]

still represents an area, even though the exact calculation needs an algebraic transformation.

Code

9.7.4 Worked example

Find

\[ \int xe^x\,dx. \]

Choose

\[ u=x, \quad \dfrac{dv}{dx}=e^x. \]

Then

\[ \dfrac{du}{dx}=1, \quad v=e^x. \]

Using integration by parts,

\[ \int xe^x\,dx = xe^x-\int e^x\,dx. \]

Therefore,

\[ \int xe^x\,dx = xe^x-e^x+C. \]

So

9.8.1 Key mathematical ideas

Further integration is about choosing from several techniques:

  • direct reverse differentiation;
  • substitution;
  • partial fractions;
  • integration by parts;
  • trigonometric identities;
  • area formulae;
  • volume of revolution formulae.

\[ A=\int_a^b y\,dx. \]

For volume generated by rotating the region under \(y=f(x)\) about the \(x\)-axis,

\[ V=\pi\int_a^b y^2\,dx. \]

9.8.3 Worked example: area

Find the exact area under

\[ y=x\sin x \]

from \(x=0\) to \(x=\pi\).

The area is

\[ A=\int_0^\pi x\sin x\,dx. \]

Using integration by parts with

\[ u=x, \quad \dfrac{dv}{dx}=\sin x, \]

we get

\[ \dfrac{du}{dx}=1, \quad v=-\cos x. \]

Therefore,

\[ \int x\sin x\,dx = -x\cos x+\int \cos x\,dx. \]

So

\[ \int x\sin x\,dx = -x\cos x+\sin x+C. \]

Hence,

\[ A= \left[-x\cos x+\sin x\right]_0^\pi. \]

Therefore,

\[ A=\pi. \]

9.8.5 Worked example: volume of revolution

The region under

\[ y=\sqrt{x} \]

from \(x=0\) to \(x=4\) is rotated completely about the \(x\)-axis. Find the exact volume.

Using

\[ V=\pi\int_a^b y^2\,dx, \]

\[ V=\pi\int_0^4 (\sqrt{x})^2\,dx. \]

  1. Let \(y=\tan^{-1}x\). Then \(\tan y=x\). Differentiating implicitly gives \[ \sec^2y\dfrac{dy}{dx}=1. \] Hence \[ \dfrac{dy}{dx}=\dfrac{1}{\sec^2y}. \] Since \(\sec^2y=1+\tan^2y=1+x^2\), we get \[ \dfrac{dy}{dx}=\dfrac{1}{1+x^2}. \]

  3. Using integration by parts, \[ \int_0^1 xe^x\,dx = \left[xe^x\right]_0^1-\int_0^1 e^x\,dx = e-(e-1)=1. \] It is not a direct reverse-chain-rule integral because \(x\) is not a constant multiple of the derivative of \(e^x\).

  4. Using integration by parts, \[ I_n=\int_0^\pi x\sin(nx)\,dx = \left[-\dfrac{x\cos(nx)}{n}\right]_0^\pi + \dfrac{1}{n}\int_0^\pi \cos(nx)\,dx. \] The second integral is zero. Therefore, \[ I_n=-\dfrac{\pi\cos(n\pi)}{n} = \dfrac{(-1)^{n+1}\pi}{n}. \]

  5. The volume is \[ V=\pi\int_0^a (\sqrt{x})^2\,dx = \pi\int_0^a x\,dx = \dfrac{\pi a^2}{2}. \] If \(V=8\pi\), then \[ \dfrac{\pi a^2}{2}=8\pi. \] Hence \(a^2=16\), and since \(a>0\), \(a=4\).

## 8.1 Derivative of $\tan^{-1}x$

### Key mathematical ideas

The derivative of the inverse tangent function is

$$
\dfrac{d}{dx}\left(\tan^{-1}x\right)
=
\dfrac{1}{1+x^2}.
$$

Using the chain rule,

$$
\dfrac{d}{dx}\left(\tan^{-1}(f(x))\right)
=
\dfrac{f'(x)}{1+[f(x)]^2}.
$$

In particular,

$$
\dfrac{d}{dx}\left(\tan^{-1}(ax+b)\right)
=
\dfrac{a}{1+(ax+b)^2}.
$$

The curve

$$

$$

is increasing, but it gradually flattens out. The derivative

$$

$$

explains this: the gradient is positive for all $x$, but tends to $0$ as $|x|$ becomes large.

## 8.2 Integration of $\dfrac{1}{x^2+a^2}$

### Key mathematical ideas

Since

$$
\dfrac{d}{dx}\left(\tan^{-1}x\right)=\dfrac{1}{1+x^2},
$$

we have

$$
\int \dfrac{1}{1+x^2}\,dx=\tan^{-1}x+C.
$$

More generally,

$$
\int \dfrac{1}{x^2+a^2}\,dx
=
\dfrac{1}{a}\tan^{-1}\left(\dfrac{x}{a}\right)+C,
\quad a>0.
$$

This result is particularly useful when a quadratic denominator cannot be factorised over the real numbers.

## 8.3 Integration of $\dfrac{kf'(x)}{f(x)}$

### Key mathematical ideas

If $f(x)$ is differentiable, then

$$
\dfrac{d}{dx}\left(\ln|f(x)|\right)
=
\dfrac{f'(x)}{f(x)}.
$$

Therefore,

$$
\int \dfrac{f'(x)}{f(x)}\,dx
=
\ln|f(x)|+C.
$$

More generally,

$$
\int \dfrac{kf'(x)}{f(x)}\,dx
=
k\ln|f(x)|+C.
$$

The key is to recognise that the numerator is a constant multiple of the derivative of the denominator.

### Visualisation using R

  geom_hline(yintercept = 0) +

### Worked example

Find

$$
\int \dfrac{6x+4}{3x^2+4x+1}\,dx.

Then

$$

Therefore,

$$
\int \dfrac{6x+4}{3x^2+4x+1}\,dx

$$

## 8.4 Integration by substitution

### Key mathematical ideas

Integration by substitution is the reverse of the chain rule.

If

$$
u=f(x),
$$

then

$$
\dfrac{du}{dx}=f'(x),
$$

so

$$
du=f'(x)\,dx.
$$

The purpose of substitution is to rewrite the whole integral in terms of a new variable.

A typical pattern is

$$
\int f'(x)[f(x)]^n\,dx.
$$

Using

$$
u=f(x),
$$

this becomes

$$
\int u^n\,du.
$$

### Visualisation using R

```{r}
#| label: fig-substitution-area
#| fig-cap: "The shaded area represents an integral suited to substitution."
#| fig-width: 6
#| fig-height: 4

)

  geom_area(alpha = 0.3) +
  geom_line(linewidth = 1.1) +
  geom_hline(yintercept = 0) +
  labs(
    title = "A substitution structure",
    subtitle = "For 2x(1 + x^2)^2, use u = 1 + x^2.",
    x = "x",
    y = "y"
  ) +
  book_theme()
```

### Worked example

Find

$$
\int 2x(1+x^2)^4\,dx.
$$

Let

$$
u=1+x^2.
$$

Then

$$
\dfrac{du}{dx}=2x,
$$

so

$$
du=2x\,dx.
$$

Therefore,

$$
\int 2x(1+x^2)^4\,dx
=
\int u^4\,du.
$$

Hence,

$$
\int u^4\,du=\dfrac{u^5}{5}+C.
$$

Substituting back gives

$$
\int 2x(1+x^2)^4\,dx
=
\dfrac{(1+x^2)^5}{5}+C.
$$

## 8.5 The use of partial fractions in integration

### Key mathematical ideas

Partial fractions are useful in integration because they split a complicated rational function into simpler terms.

For example,

$$
\dfrac{1}{x(x+1)}
=
\dfrac{1}{x}-\dfrac{1}{x+1}.
$$

Therefore,

$$
\int \dfrac{1}{x(x+1)}\,dx
=
\ln|x|-\ln|x+1|+C.
$$

::: {.callout-warning}
Before using partial fractions, check whether the algebraic fraction is proper. If the degree of the numerator is greater than or equal to the degree of the denominator, do long division first.
:::

### Visualisation using R

```{r}

#| fig-cap: "A rational function split into partial fractions."
#| fig-width: 6
#| fig-height: 4

  x = seq(0.2, 5, length.out = 800)
)

df$original <- 1 / (df$x * (df$x + 1))
df$partial_fraction <- 1 / df$x - 1 / (df$x + 1)

  geom_line(aes(y = original, linetype = "original"), linewidth = 1.1) +
  geom_line(aes(y = partial_fraction, linetype = "partial fractions"), linewidth = 1.1) +
  geom_hline(yintercept = 0) +
  labs(
    title = "Partial fractions before integration",
    subtitle = "The two expressions are identical on their common domain.",
    x = "x",
    y = "value",
    linetype = "Expression"
  ) +
  book_theme()
```

### Worked example

Find

$$
\int \dfrac{3x+5}{(x+1)(x+2)}\,dx.
$$

First write

$$
\dfrac{3x+5}{(x+1)(x+2)}
=
\dfrac{A}{x+1}+\dfrac{B}{x+2}.
$$

Multiplying throughout by $(x+1)(x+2)$ gives

$$
3x+5=A(x+2)+B(x+1).
$$

So

$$
3x+5=(A+B)x+(2A+B).
$$

Equating coefficients gives

$$
A+B=3
$$

and

$$
2A+B=5.
$$

Solving gives

$$
A=2,
\quad
B=1.
$$

Therefore,

$$
\int \dfrac{3x+5}{(x+1)(x+2)}\,dx
=
\int \left(\dfrac{2}{x+1}+\dfrac{1}{x+2}\right)\,dx.
$$

Hence,

$$
\int \dfrac{3x+5}{(x+1)(x+2)}\,dx
=
2\ln|x+1|+\ln|x+2|+C.
$$

## 8.6 Integration by parts

### Key mathematical ideas

Integration by parts is the reverse of the product rule.

The product rule says

$$
\dfrac{d}{dx}(uv)=u\dfrac{dv}{dx}+v\dfrac{du}{dx}.
$$

Rearranging and integrating gives

$$
\int u\dfrac{dv}{dx}\,dx
=
uv-\int v\dfrac{du}{dx}\,dx.
$$

This is usually written as

$$
\int u\,dv=uv-\int v\,du.
$$

### Why display integration by parts?

A graph does not show the algebraic mechanism of integration by parts as directly as it shows area or volume. Its value is different: it shows the integral as an area under a product curve, while the method rewrites that area in terms of a boundary term minus a simpler area.

So integration by parts is worth displaying when we want students to see that an apparently difficult product integral such as

$$
\int xe^x\,dx
$$

still represents an area, even though the exact calculation needs an algebraic transformation.

### Visualisation using R

```{r}

#| fig-cap: "The shaded area represents the integral of x e^x from 0 to 1."
#| fig-width: 6
#| fig-height: 4

  x = seq(0, 1, length.out = 600)
)

  geom_area(alpha = 0.3) +
  geom_line(linewidth = 1.1) +
  geom_hline(yintercept = 0) +
  labs(
    title = "Area under y = x e^x",
    subtitle = "Integration by parts gives the exact area from 0 to 1 as 1.",
    x = "x",
    y = "y"
  ) +
  book_theme()
```

### Worked example

Find

$$
\int xe^x\,dx.
$$

Choose

$$
u=x,
\quad
\dfrac{dv}{dx}=e^x.
$$

Then

$$
\dfrac{du}{dx}=1,
\quad
v=e^x.
$$

Using integration by parts,

$$
\int xe^x\,dx
=
xe^x-\int e^x\,dx.
$$

Therefore,

$$
\int xe^x\,dx
=
xe^x-e^x+C.
$$

So

$$
\int xe^x\,dx
=
(x-1)e^x+C.
$$

## 8.7 Further integration

### Key mathematical ideas

Further integration is about choosing from several techniques:

- direct reverse differentiation;
- substitution;
- partial fractions;
- integration by parts;
- trigonometric identities;
- area formulae;
- volume of revolution formulae.

For area under a curve above the $x$-axis,

$$
A=\int_a^b y\,dx.
$$

For volume generated by rotating the region under $y=f(x)$ about the $x$-axis,

$$
V=\pi\int_a^b y^2\,dx.
$$

### Visualisation using R: shaded area

```{r}

  geom_line(linewidth = 1.1) +
  geom_hline(yintercept = 0) +
  labs(
    title = "Area under y = x sin(x)",
    subtitle = "The exact area from 0 to pi is pi.",
    x = "x",
    y = "y"
  ) +
  book_theme()
```

### Worked example: area

Find the exact area under

$$
y=x\sin x
$$

from $x=0$ to $x=\pi$.

The area is

$$
A=\int_0^\pi x\sin x\,dx.
$$

Using integration by parts with

$$
u=x,
\quad
\dfrac{dv}{dx}=\sin x,
$$

we get

$$
\dfrac{du}{dx}=1,
\quad
v=-\cos x.
$$

Therefore,

$$
\int x\sin x\,dx
=
-x\cos x+\int \cos x\,dx.
$$

So

$$
\int x\sin x\,dx
=
-x\cos x+\sin x+C.
$$

Hence,

$$
A=
\left[-x\cos x+\sin x\right]_0^\pi.
$$

Therefore,

$$
A=\pi.
$$

### Worked example: volume of revolution

The region under

$$
y=\sqrt{x}
$$

from $x=0$ to $x=4$ is rotated completely about the $x$-axis. Find the exact volume.

Using

$$
V=\pi\int_a^b y^2\,dx,
$$

we get

$$
V=\pi\int_0^4 (\sqrt{x})^2\,dx.
$$

So

$$
V=\pi\int_0^4 x\,dx.
$$

Therefore,

$$
V=\pi\left[\dfrac{x^2}{2}\right]_0^4.
$$

Hence,

$$

$$

### Answers to STEP-style extension questions

1. Let $y=\tan^{-1}x$. Then $\tan y=x$. Differentiating implicitly gives
   $$
   \sec^2y\dfrac{dy}{dx}=1.
   $$
   Hence
   $$
   \dfrac{dy}{dx}=\dfrac{1}{\sec^2y}.
   $$
   Since $\sec^2y=1+\tan^2y=1+x^2$, we get
   $$
   \dfrac{dy}{dx}=\dfrac{1}{1+x^2}.
   $$

2. Since
   $$
   \int \dfrac{1}{1+x^2}\,dx=\tan^{-1}x+C,
   $$
   we have
   $$
   \int_0^1 \dfrac{1}{1+x^2}\,dx
   =
   \left[\tan^{-1}x\right]_0^1
   =
   \dfrac{\pi}{4}.
   $$
   This is the exact area under $y=\dfrac{1}{1+x^2}$ from $0$ to $1$.

3. Using integration by parts,
   $$
   \int_0^1 xe^x\,dx
   =
   \left[xe^x\right]_0^1-\int_0^1 e^x\,dx
   =
   e-(e-1)=1.
   $$
   It is not a direct reverse-chain-rule integral because $x$ is not a constant multiple of the derivative of $e^x$.

4. Using integration by parts,
   $$
   I_n=\int_0^\pi x\sin(nx)\,dx
   =
   \left[-\dfrac{x\cos(nx)}{n}\right]_0^\pi
   +
   \dfrac{1}{n}\int_0^\pi \cos(nx)\,dx.
   $$
   The second integral is zero. Therefore,
   $$
   I_n=-\dfrac{\pi\cos(n\pi)}{n}
   =
   \dfrac{(-1)^{n+1}\pi}{n}.
   $$

5. The volume is
   $$
   V=\pi\int_0^a (\sqrt{x})^2\,dx
   =
   \pi\int_0^a x\,dx
   =
   \dfrac{\pi a^2}{2}.
   $$
   If $V=8\pi$, then
   $$
   \dfrac{\pi a^2}{2}=8\pi.
   $$
   Hence $a^2=16$, and since $a>0$, $a=4$.
